# homey-meshdriver

## Introduction
This module is used to make the creation of Z-Wave & ZigBee apps easier.

It is essentially a map-tool from Homey-capabilities to Z-Wave Command Classes or ZigBee endpoints.

## Installation

```bash
$ npm install homey-meshdriver
```
